KEEP THESE THINGS IN MIND

There are a few things you need to be aware of, before you start using the mod manager:

  1. DO NOT TOUCH the manifest.xml file!
    It contains crucial staging information about your installed mods.

  2. DO NOT CHANGE the name of any folder in the Fallout76\Mods folder.
    The name of these folders are used to identify the mod.
    If you don't know what mod this is (e.g. the folder is called "Version 1.2.3.4" with no name attached to it),
    just open the "Manage mods" window, select a mod and click on "Open folder".
    You can also change the mod's title to your liking.

  3. DO NOT DELETE any folder in the Fallout76\Mods folder.
    If you want to delete the mod, just open the "Manage mods" window, select a mod, click "Delete mod" and then "Deploy".

  4. You may change any of the files within a Fallout76\Mods\<your mod> folder.
Breaking these "rules" can cause issues, and this is why I won't help you, if you do so.


FAQ

How to completely disable mods?

Unless you have installed mods manually or with a different tool,
it should be as simple as unchecking all mods and clicking "Deploy".
This will remove all mods from the game and *.ini files.


What happens to mods I've installed manually?

The tool won't touch these mods. If you want to manage them, you have to import them first.


Can I get banned for using mods?

Yes, of course. In fact, you can get permanently banned.


Why don't you distribute Archive2 with your tool?

Because Bethesda owns the Archive2 tool and if I'd do so, I'd basically pirate it.
Other mod authors might do this, but I don't want any legal issues.


ABOUT ARCHIVE2

In order to use the mod manager, you need to install Archive2.
Archive2 is the official tool to pack/extract *.ba2 archives.
It is shipped with Fallout 4's Creation Kit.
Note that Skyrim's Creation Kit won't cut it, you need Fallout 4.

I'm not going to distribute Archive2 with my tool because I don't want any legal issues. If you choose to pirate it, that's your pigeon.

How to install Archive2:

  1. Install Fallout 4.
  2. Get the Bethesda.net Launcher, create a account if needed.
  3. Install "Creation Kit: Fallout 4" (see Figure 1)
  4. Open the "Manage mods" window and set the path to "Archive2.exe". (see Figure 2)
    You can find Archive2.exe in <your steam folder>\steamapps\common\Fallout 4\Tools\Archive2\Archive2.exe.
